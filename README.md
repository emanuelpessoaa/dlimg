# Dlimg

> Download all images from src attribute within img tag by type

## Install

```bash
npm install dlimg -g
```

## Usage CLI
##### To download all image content-type:

```bash
dlimg http://www.whatever.com
```
##### To filter by type:

```bash
dlimg http://www.whatever.com --type jpg
```
